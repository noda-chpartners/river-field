# River Field - 占いカフェ ランディングページ

## 1. プロジェクト概要
豊田市にある占いカフェ「River Field」の1ページ完結型LP。深い紺色を基調とした和風で落ち着いたデザイン。スクロールアニメーションでゆったりとした体験を提供し、占術メニュー・鑑定メニュー・アクセス情報を伝える。

## 2. ページ構造
- `/` - トップページ（シングルページLP）

## 3. コア機能
- [x] Hero セクション（ブランド名・キャッチコピー・CTA）
- [x] About / 店舗紹介セクション
- [x] 占術紹介セクション（カードグリッド）
- [x] メニュー・料金セクション
- [x] アクセス・営業情報セクション
- [x] フッター
- [x] スクロール連動フェードインアニメーション

## 4. データモデル設計
データベース不要（静的LP）

## 5. バックエンド / サードパーティ連携計画
- Supabase: 不要
- Shopify: 不要
- Stripe: 不要

## 6. 開発フェーズ計画

### フェーズ1: LP全体の構築
- 目標: 全セクションを含む1ページLPを完成させる
- 成果物: スクロールアニメーション付きの完成LP
